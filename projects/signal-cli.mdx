## The idea

Signal CLI explores how much polish a terminal workflow can carry before it stops feeling lightweight. The target was a tool that felt fast, readable, and calm under pressure.

## Key choices

- Clear hierarchy between headings, values, and command hints
- Compact output that still scans well in noisy logs
- A design language that respects terminal constraints instead of fighting them
